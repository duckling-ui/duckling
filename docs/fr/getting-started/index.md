# Bien démarrer

Cette section couvre l’installation et l’exécution de Duckling.

## Contenu

- Installation
- Démarrage rapide
- Docker



    C'est tout ! Python ou Node.js ne sont pas requis.

=== "Développement local"

    - **Python 3.10+** (3.13 recommandé)
    - **Node.js 18+**
    - **npm ou yarn**
    - **Git**
